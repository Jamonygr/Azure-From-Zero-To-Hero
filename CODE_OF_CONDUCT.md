# Community Standards

Azure From Zero To Hero exists to help people learn Azure and Terraform through practical, repeatable labs. Keep the community professional, generous, and focused on helping learners build confidence.

## Expected Behavior

- Be respectful when explaining mistakes or suggesting changes.
- Assume learners may be new to Azure, Terraform, PowerShell, Git, or GitHub.
- Share context, commands, screenshots, and references that help others reproduce an issue.
- Keep feedback about the work, not the person.
- Protect secrets, subscription details, tenant details, and private infrastructure names.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Posting credentials, tenant data, or private customer information.
- Pressuring people to run destructive commands without clear cleanup context.
- Dismissing beginner questions instead of redirecting them to the right lesson or guide.

## Reporting

For behavior concerns, open a GitHub issue only when the report can be public and does not include private details. For sensitive concerns, use GitHub's private reporting options when available on the repository.

Project maintainers may edit, hide, lock, or remove content that makes the learning space unsafe or unhelpful.
